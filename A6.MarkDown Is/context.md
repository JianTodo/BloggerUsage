## 前言
**MD 副檔名的檔案 是什麼呢？**
> MD 副檔名全名為 [Markdown](https://en.wikipedia.org/wiki/Markdown)，
	是一種「輕量級」標記式語言，
	終極目標就是讓**讀寫**更加方便。

## 標題
- 範例
	<pre class="brush:sehll;">
	// H1 文字大小
	# Size H1
	// H2 文字大小
	## Size H2
	// H6 文字大小
	###### Size H6
	</pre>
- 結果
	# Size H1

	## Size H2

	###### Size H6
	
## 文字
- 範例
	<pre class="brush:sehll;">
	// 斜體
	_emphasize_
	// 粗體
	*Strong*
	// 黃色字體
	==Yellow text.==
	// 刪除體
	~~Mistaken text.~~
	// 區塊
	> Quoted text.
	</pre>
- 結果
	_emphasize_
	**Strong**  
	==Yellow text.==
	~~Mistaken text.~~
	> Quoted text.
